## 概要

Firebase Authentication により Google アカウントでログイン / ログアウトできる。未ログイン時は記録機能を使えない。

## 詳細仕様 / 振る舞い

- ログイン画面に「Google でログイン」ボタンを表示する。
- ログイン成功後、初回はユーザー情報（uid・表示名・メール）を登録する。
- ログアウトするとログイン画面に戻る。
- 未ログインで保護ページにアクセスした場合はログイン画面へリダイレクトする。

## 受け入れ基準

- [ ] Google アカウントでログインできる。
- [ ] ログアウトできる。
- [ ] 未ログイン状態では記録機能にアクセスできない。
