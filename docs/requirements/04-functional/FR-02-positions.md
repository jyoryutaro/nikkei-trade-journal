## 概要

チャート上で選択した時点に、ポジション記録（売買方向・種別・金額）またはコメントのみの記録を登録する。記録は `journal_entries` として永続化する。

## 詳細仕様 / 振る舞い

- チャートをクリックして対象時点（足）を選択する。
- 入力フォーム:
  - **売買**（ラジオ）: 未選択 / 買い(long) / 売り(short)
  - **種別**（ラジオ、買い/売り選択時のみ表示）: 新規(open) / 決済(close)
  - **金額**（数値、買い/売り選択時のみ表示）
  - **コメント**（テキスト、常時表示）
- バリデーション:
  - 未選択（コメントのみ）→ コメントが非空であれば登録可。
  - 買い / 売り → 種別の選択 **かつ** 正の金額が必須（全て入力するまで送信不可）。コメントは任意。
  - 同じ検証をバックエンド（`journal.NewEntry`）でも行い、不正入力は API が 400 で拒否する。
- 設計・フロー（mermaid）は[設計書 03-position-recording](../../design/03-position-recording.md)を参照。

## 受け入れ基準

- [x] チャート上の時点に売買・種別・金額を登録できる。
- [x] コメントのみでも登録できる。
- [x] 必須項目が未入力のときは送信できない（フロント／API 双方で検証）。
- [x] 再読込後も保持される（MySQL に永続化）。
- [x] 登録内容がチャート上にマーカー表示される（(時点, 金額) 座標にマーカー、ホバーで吹き出し表示）。
