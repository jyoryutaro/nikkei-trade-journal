## 概要

ポジションや任意の時点にメモ・タグを付与する。

## 詳細仕様 / 振る舞い

- ポジション（Trade）またはチャート上の任意時点にコメントを紐づける。
- 本文に加えてタグを付けられる。
- コメントは編集・削除できる。

## 受け入れ基準

- [ ] ポジション / 時点にコメントを追加できる。
- [ ] コメントを編集・削除できる。
- [ ] 再読込後も保持される。
