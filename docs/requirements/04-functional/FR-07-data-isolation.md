# FR-07 データ分離

| 項目 | 内容 |
| --- | --- |
| ID | FR-07 |
| 機能名 | データ分離 |
| 優先度 | Must |
| 担当 | <担当者> |
| ステータス | ドラフト |

## 概要

ログインユーザー本人の記録のみ閲覧・編集でき、他ユーザーのデータにアクセスできない。

## 詳細仕様 / 振る舞い

- Firestore セキュリティルールで `request.auth.uid == uid` のドキュメントのみ読み書き可能とする。
- クライアント側のクエリも自身の uid に限定する。

## 受け入れ基準

- [ ] 別ユーザーでログインすると自分の記録のみ表示される。
- [ ] 他ユーザーのデータを直接参照・編集できない（セキュリティルールで拒否）。
