## 概要

日経225先物のローソク足 / ラインチャートを期間切替（日足 / 分足等）で表示する。

## 詳細仕様 / 振る舞い

- 以下の足種を切り替えられる。

  | 足種 | 識別子 | 備考 |
  | --- | --- | --- |
  | 1分足 | `1m` | DB に直接保存するベース足 |
  | 5分足 | `5m` | 1m を集計して生成 |
  | 30分足 | `30m` | 1m を集計して生成 |
  | 1時間足 | `1h` | 1m を集計して生成 |
  | 日足 | `1d` | 1m を集計して生成（UTC 日単位） |

- 5分足以上はバックエンドが `market_data (timeframe='1m')` を読み取り、OHLCV を集計してレスポンスを返す（DB に派生足を保存しない）。
- 集計バケットは Unix エポックからの経過秒数で整列する（例: 5m バケットは :00/:05/:10 … 刻み）。
- ズーム / スクロールで時間軸を移動できる。
- 時間窓ボタン（足種に応じて変化）でワンクリックで直近 N 分/時間に絞り込める。
- 対象限月はメジャーSQ（3/6/9/12月限）の各限月を対象とし、取得可能な範囲を表示する（[12-open-issues.md](../12-open-issues.md) Q-05）。
- 商品種別は限定せず、データ取得先に依存する（[12-open-issues.md](../12-open-issues.md) Q-06）。
- 価格データの取得元は Yahoo Finance（`NKD=F`）。取得・保存仕様は[要件定義 UC-01](../../spec/use-cases/UC-01-fetch-market-data.md)を参照（[12-open-issues.md](../12-open-issues.md) Q-01）。

## 受け入れ基準

- [x] 日経225先物のチャートが表示される。
- [x] 足種を切り替えると表示が更新される（1m / 5m / 30m / 1h / 1d）。
- [x] 時間窓ボタンで表示範囲を絞り込める。
- [ ] 日足は JST 日付単位で集計される（現在は UTC 日単位、要確認）。
