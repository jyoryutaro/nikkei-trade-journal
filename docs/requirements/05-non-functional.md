# 5. 非機能要件（NFR）

| 区分 | 要件 |
| --- | --- |
| パフォーマンス | <例：チャート描画は1秒以内> |
| 対応環境 | PCブラウザ（Chrome / Safari / Edge の最新版）。モバイル対応の要否は TBD |
| 可用性 / データ保持 | 記録データは Firestore に保存し、端末を問わずログインすれば参照できる。消失しないこと（バックアップ方針は TBD） |
| セキュリティ / プライバシー | 通信は HTTPS。認証は Firebase Authentication（Google）。Firestore セキュリティルールで各ユーザーの記録は本人のみアクセス可能とし、第三者に共有しない |
| 国際化 | <例：日本語のみ> |
