# 5. 非機能要件（NFR）

| 区分 | 要件 |
| --- | --- |
| パフォーマンス | <例：チャート描画は1秒以内> |
| 対応環境 | PCブラウザ（Chrome / Safari / Edge の最新版）。モバイル対応の要否は TBD |
| 可用性 / データ保持 | 記録データは MySQL に保存する。消失しないこと（バックアップ方針は TBD）。当面はローカル環境で稼働 |
| セキュリティ / プライバシー | 当面はローカル単一ユーザーのため認証なし。マルチユーザー化時に認証・データ分離・通信の暗号化（HTTPS）を導入する（[12-open-issues.md](12-open-issues.md) Q-03） |
| 国際化 | <例：日本語のみ> |
