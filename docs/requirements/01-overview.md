# 1. 概要 / 背景

- **目的**: 日経225先物（日経平均先物）のチャート上で、自身のポジションの IN / OUT とコメントを記録・振り返りできるようにする。
- **提供形態**: Webアプリケーション。フロントエンド（React + TypeScript + Vite）＋ バックエンド（Go）＋ DB（MySQL）の自前構成。データは MySQL に保存する。当面はローカル単一ユーザーで利用し、認証は導入しない（マルチユーザー化・認証は将来課題。[12-open-issues.md](12-open-issues.md) Q-03）。技術選定の詳細は[設計書](../design/01-tech-stack.md)を参照。
- **背景・課題**: <なぜ作るのか。現状どんな不便があるか。例：売買タイミングの根拠や反省が手元に残らず、トレードの再現性が上がらない>
- **期待する効果**: <例：エントリー/エグジット理由を蓄積し、勝ち負けの傾向を可視化して改善につなげる>
