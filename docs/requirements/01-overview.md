# 1. 概要 / 背景

- **目的**: 日経平均（日経225）のチャート上で、自身のポジションの IN / OUT とコメントを記録・振り返りできるようにする。
- **提供形態**: Webアプリケーション。バックエンド基盤に Firebase を採用し、認証は Firebase Authentication（Google プロバイダ）、データ保存は Firestore（クラウド）で管理する。ユーザーは Google アカウントでログインして自分の記録にアクセスする。
- **背景・課題**: <なぜ作るのか。現状どんな不便があるか。例：売買タイミングの根拠や反省が手元に残らず、トレードの再現性が上がらない>
- **期待する効果**: <例：エントリー/エグジット理由を蓄積し、勝ち負けの傾向を可視化して改善につなげる>
