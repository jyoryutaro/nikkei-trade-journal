# 2. スコープ

## 2.1 対象に含むもの（In Scope）

- 日経225先物のチャート表示
- チャート上でのポジション IN / OUT の記録
- 各ポジション・各時点へのコメント付与
- 記録データの MySQL 保存

## 2.3 当面スコープ外（将来課題）

- ユーザー認証（当面はローカル単一ユーザー。マルチユーザー化時に検討。[12-open-issues.md](12-open-issues.md) Q-03）
- ユーザーごとのデータ分離（同上）

## 2.2 対象に含まないもの（Out of Scope）

- <例：実際の発注・約定（証券会社連携）>
- <例：日経225先物以外の銘柄（現物指数・個別株・他の先物など）>
- <例：自動売買・シグナル生成>
