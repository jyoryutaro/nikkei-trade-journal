## 4. 機能要件（FR）

> 優先度: Must（必須） / Should（推奨） / Could（任意）

| ID | 機能 | 説明 | 優先度 |
| --- | --- | --- | --- |
| FR-01 | チャート表示 | 日経平均のローソク足/ラインチャートを期間切替（日足/分足等）で表示する | Must |
| FR-02 | ポジション記録 | チャート上の任意の時点に IN / OUT を登録する（価格・数量・ロング/ショート） | Must |
| FR-03 | コメント記録 | ポジションや任意の時点にメモ・タグを付与する | Must |
| FR-04 | 一覧/編集 | 記録したトレードを一覧表示・編集・削除する | Should |
| FR-05 | 損益表示 | IN/OUT から損益（pips/金額/%）を自動計算して表示する | Should |
| FR-06 | 集計/振り返り | 勝率・損益推移などを集計表示する | Could |
