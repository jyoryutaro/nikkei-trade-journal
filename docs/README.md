# ドキュメント

本プロジェクトのドキュメントは、抽象度（何を作るか → どう作るか）に応じて3階層に分けています。

| 階層 | 内容 | 場所 |
| --- | --- | --- |
| 要求定義 | 何を実現したいか（目的・スコープ・機能/非機能要件など） | [requirements/](requirements/README.md) |
| 要件定義 | システムとして満たすべき振る舞いの仕様（ユースケース等） | [spec/](spec/README.md) |
| 設計書 | どう実現するか（技術選定・アーキテクチャ等） | [design/](design/README.md) |
